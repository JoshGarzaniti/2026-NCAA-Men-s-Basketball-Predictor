# 2026-NCAA-Men-s-Basketball-Predictor
Vibecoding a Gradient Boosted Random Forest Model to predict the March Madness Winner

To Use this codespace properly:
Feed the 2002-2025 data of the NCAA basketball data document into the March_Madness_26 python file. Use that to test the model.

Then import the 2026 data from the NCAA basketball data document into the March_Madness_26_Prediction python file for the actual 2026 predicted tournament.
